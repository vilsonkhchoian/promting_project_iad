# Essay Checker Bot (promting_project_IAD)

Telegram-бот для проверки эссе с помощью GigaChat.

Бот принимает задание, затем текст эссе, отправляет их в GigaChat и возвращает оценку с пояснениями.

## Возможности

- прием задания для эссе
- прием текста эссе
- отправка запроса в GigaChat
- получение оценки и пояснений
- поддержка длинных ответов
- команда отмены проверки

## Команды

| Команда | Описание |
|---|---|
| `/start` | Начать новую проверку |
| `/cancel` | Отменить текущую проверку |

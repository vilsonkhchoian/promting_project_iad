# Essay Checker Bot

Telegram-бот для автоматической проверки эссе по английскому языку с помощью GigaChat. 
Проект ориентирован на проверку письменной части независимого экзамена НИУ ВШЭ по английскому языку. Пользователь отправляет задание и текст эссе, после чего бот возвращает оценку и комментарий по критериям.

## Возможности

- прием задания для эссе
- прием текста эссе
- автоматическая проверка через GigaChat
- выставление оценки по шкале от 0 до 10
- комментарий с пояснением ошибок
- поддержка длинных ответов
- команда отмены проверки

## Команды

| Команда | Описание |
|---|---|
| `/start` | Начать новую проверку |
| `/cancel` | Отменить текущую проверку |

## Основная идея проекта

Проект решает проблему автоматической проверки письменных работ по английскому языку. Студент может отправить эссе и получить быструю оценку по критериям, похожим на критерии независимого экзамена НИУ ВШЭ.

Оценивание строится по следующим критериям:

- Task Response
- Coherence and Cohesion
- Lexical Resource and Register
- Grammatical Range and Accuracy
- Register

## Используемые модели

В проекте тестировались разные языковые модели:

- GigaChat 2 Lite
- GigaChat 2 Max
- OpenAI GPT free
- Trinity
- Nvidia Nemotron 3

Главная цель сравнения заключалась в подборе наиболее точного промта для проверки эссе.

## Промты

Для проекта было подготовлено несколько вариантов промтов:

| Промт | Описание |
|---|---|
| Промт 1 | Подробный промт по схеме КОМПОЗИТОР с ролью, миссией, примерами, критериями и форматом ответа. |
| Промт 2 | Простой промт, где модели передаются критерии и просьба оценить эссе. |
| Промт 3 | Пошаговый промт, в котором модель последовательно анализирует эссе по критериям. |
| Промт 4 | Минимальный промт с критериями, заданием и ответом студента. |
| Промт 5 | Концептуальный промт, где модель ориентируется на примеры эссе и названия критериев. |
| Промт 6 | Улучшенный лучший (из предыдущих промтов) промт с защитой от prompt-инъекций. |

## Метрики качества

Для сравнения моделей и промтов использовались метрики:

- MAE
- RMSE
- MAPE
- SMAPE
- Bias
- Within_1
- Within_2
- Spearman
- QWK
- Leader_Score

Эти метрики помогают оценить точность выставления баллов, устойчивость модели, наличие завышения или занижения оценок и близость к реальным оценкам преподавателя.
